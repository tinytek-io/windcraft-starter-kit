# File trigger

File triggers are used to trigger a file selection event. This is useful for uploading files or selecting files for other purposes.

```tsx preview
import { FileTrigger } from "react-aria-components"
import { Button } from '@/ui/components/Button';

export default function Example() {
  return (
    <FileTrigger>
      <Button>Select a file</Button>
    </FileTrigger>
  );
}
```
