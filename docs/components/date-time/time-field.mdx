# TimeField

`TimeField` is a component that allows users to input a time value. It is a wrapper around the `TextField` component with additional functionality to handle time values.

```tsx
import { TimeField } from '@/ui/components/TimeField';

export default function Example() {
  return (
    <TimeField
      aria-label="Select time"
      onChange={function(){}}
      onFocusChange={function(){}}
    />
  );
}
```
