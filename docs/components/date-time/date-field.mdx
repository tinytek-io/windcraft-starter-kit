# DateField

The DateField component is a text input field that allows users to enter a date. It is used in forms to collect dates from users.

```tsx
import { DateField } from '@/ui/components/DateField';

export default function Example() {
  return (
    <DateField
      aria-label="Select date"
      onChange={function(){}}
      onFocusChange={function(){}}
    />
  );
}
```
